animations au défilement

data-reveal

Tous les presets et modes de stagger — actif si VITE_AOS=true dans .env.local.

Fade Mask Zoom Stagger Attributs

Presets Fade

Fondu avec translation — opacity: 0 → 1

fade

Opacité seule, sans déplacement.

data-reveal="fade"

fadeBlur

Fondu + dissolution du flou.

data-reveal="fadeBlur"

fadeUp défaut

Fondu + glissement vers le haut.

data-reveal="fadeUp"

fadeDown

Fondu + glissement vers le bas.

data-reveal="fadeDown"

fadeLeft

Fondu + glissement depuis la droite.

data-reveal="fadeLeft"

fadeRight

Fondu + glissement depuis la gauche.

data-reveal="fadeRight"

Presets Mask

Révélation par masque clip-path — l'opacité reste à 1.

maskUp

Du bas vers le haut.

data-reveal="maskUp"

maskDown

Du haut vers le bas.

data-reveal="maskDown"

maskLeft

De droite à gauche.

data-reveal="maskLeft"

maskRight

De gauche à droite.

data-reveal="maskRight"

Presets Zoom

Fondu + mise à l'échelle.

zoomIn

data-reveal="zoomIn"

zoomOut

data-reveal="zoomOut"

Attributs

Tous les data-reveal-* disponibles.

Attribut Valeurs Défaut Description
data-revealpreset, stagger[.asc|.desc|.random]—Preset d'animation ou mode cascade.
data-reveal-presetnom d'un presetfadeUpPreset pour le mode stagger.
data-reveal-delaynombre (s)0Délai avant le démarrage.
data-reveal-durationnombre (s)0.7Durée de l'animation.
data-reveal-staggernombre (s)0.05Intervalle entre enfants (stagger).
data-reveal-selectorsélecteur CSS> *Filtre les enfants à animer (stagger).
