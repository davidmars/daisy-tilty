2. Variantes inline

Tu peux tester du gras, de l'italique, du souligné, du surligné, du petit texte, un HTML abrégé et un extrait de code inline comme npm run dev.

Exemple de lien: lien principal. Autre lien: lien secondaire. Et un texte barré pour tester la lecture d'états: ancienne version remplacée par nouvelle version.
